# Ayatalla Eweis - Interactive Resume

This is an interactive mind map style resume for Ayatalla Eweis, showcasing her professional experience, skills, and qualifications as a Creative Designer & Marketing Manager.

## Features

- Interactive mind map visualization
- Collapsible sections for different career aspects
- AI-powered chatbot for questions about experience
- Mobile-responsive design
- Professional styling with custom color scheme

## View Online

This resume is hosted on GitHub Pages and can be viewed at: [https://aya-eweis.github.io/aya-eweis-resume/](https://aya-eweis.github.io/aya-eweis-resume/)

## Technology Stack

- HTML5
- CSS3 (with custom animations and responsive design)
- Vanilla JavaScript
- Google Gemini AI for chatbot functionality
- GitHub Pages for hosting
